# Rokhneghah Cyber Lab

An open-source cybersecurity training platform inspired by PortSwigger Web Security Academy.

Rokhneghah Cyber Lab provides interactive labs where students, developers, and bug bounty hunters can practice real-world web vulnerabilities in a safe and controlled environment.

---

## 🌐 Live Demo

https://afpropent.vercel.app/

## 🚀 Features

* Interactive cybersecurity labs
* Practice real vulnerabilities in a safe environment
* Hands-on learning experience
* Designed for bug bounty hunters and security learners
* Inspired by modern web security training platforms

---

## 🧪 Vulnerabilities Covered

The platform includes labs focused on common web security vulnerabilities:

* Cross-Site Scripting (XSS)
* Broken Access Control
* IDOR (Insecure Direct Object Reference)
* CSRF (Cross-Site Request Forgery)
* API Security Issues
* Authentication vulnerabilities

Each lab is designed to simulate real-world scenarios to help learners understand how vulnerabilities work and how they are exploited.

---

## 🎯 Project Goal

The goal of this project is to make practical cybersecurity education accessible to everyone.

Many students and aspiring security researchers cannot afford expensive training platforms. Rokhneghah Cyber Lab aims to provide a free and open learning environment where anyone can practice and improve their web security skills.

## 💻 Running Locally

Clone the repository:

```bash
git clone https://github.com/nobody1919/rokhneghah-cyber-lab
```

Go to the project directory:

```bash
cd rokhnegah-cyber-lab
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
